Guardian Security Suite

Automated Auditing & System Hardening for Linux

About Guardian

An overview of why this suite was built and what it aims to solve.

Guardian is a security-focused toolkit designed to help administrators and power users identify vulnerabilities within their Linux environment. It streamlines the process of system auditing by combining several critical security checks into a single, easy-to-run CLI interface.

Core Modules

System Auditor

The primary engine for identifying configuration risks.

This script checks for common misconfigurations, including overly-permissive file settings (SUID/SGID), exposed network ports, and unauthorized background services that could serve as entry points for attackers.

CVE Checker

Proactive vulnerability detection against known databases.

This module scans installed packages and the system kernel against a database of Common Vulnerabilities and Exposures (CVEs), providing a report on components that require immediate patching. This section is still in development.

Password Strength Tester

Ensures identity security through credential analysis.

A utility that evaluates the complexity of user passwords and system policies to ensure they meet modern entropy standards, helping prevent brute-force attacks.

Quick Start

How to deploy the suite on your local machine.

# Clone the repository
git clone https://github.com/Souldragon9912/Guardian.git

# Enter the directory
cd Guardian

# Run the installer
sudo ./install.sh

Once installed, you can launch the suite by simply typing guardian in your terminal.

Project Ecosystem

Complementary tools for Linux education.

Guardian works seamlessly alongside Linux Playground, an educational CLI tool designed to teach new users how to navigate the command line safely while Guardian ensures the environment remains secure.
